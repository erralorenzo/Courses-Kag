# Courses-Kag

Repository for course materials and Kaggle-related projects and notebooks.

## About

This repository contains notes, exercises, and example notebooks used for courses and Kaggle experiments. Use it as a personal workspace to collect learning resources, datasets pointers, and reproducible analyses.

## Contents

- notebooks/        - Jupyter notebooks and Kaggle kernels
- data/             - (Optional) small sample datasets or pointers to external data
- exercises/        - Course exercises and solutions
- docs/             - Documentation and supplementary material

(If these folders don't exist yet, create them as needed.)

## How to use

1. Clone the repository:

   git clone https://github.com/erralorenzo/Courses-Kag.git

2. Create and activate a virtual environment, then install dependencies (if any):

   python -m venv .venv
   source .venv/bin/activate  # macOS / Linux
   .venv\Scripts\activate     # Windows

3. Open notebooks in Jupyter or VS Code and run cells.

## Contributing
